Приложение было сделано на React Native и Expo

1. Установка зависимостей

   ```bash
   npm install
   ```

2. Настройка окружения

Создайте .env файл и добавьте в него переменные EXPO_PUBLIC_API_KEY и EXPO_PUBLIC_API_URL

Например:

  ```bash
   EXPO_PUBLIC_API_KEY=31kwzxnpb98adng3
   EXPO_PUBLIC_API_URL=https://api.sstats.net
   ```


3. Запуск приложения

   ```bash
   npx expo start
   ```

Можно запустить в браузере:

   ```bash
   npm run web
   ```
